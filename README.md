# Shantanu Pal — Clinical Research Coordinator Portfolio

A premium futuristic healthcare portfolio website.

## 🚀 GitHub Pages Deployment

1. Create a new GitHub repository (e.g. `shantanu-portfolio`)
2. Upload all files in this folder:
   - `index.html`
   - `style.css`
   - `script.js`
   - `Gemini_Generated_Image_vlv2lovlv2lovlv2.png` (profile photo)
   - `Use_the_third_202602192323_8f91g.mp4` (intro video)
   - `SHANTANU_PAL-CV-CRC.pdf` (CV download)
3. Go to **Settings → Pages → Source: Deploy from branch → main → / (root)**
4. Your site will be live at `https://<username>.github.io/shantanu-portfolio/`

## 📁 File Structure
```
shantanu-portfolio/
├── index.html   — Main page (single-page portfolio)
├── style.css    — All styles, glassmorphism, animations
├── script.js    — Three.js DNA, GSAP, interactivity
├── *.png        — Profile photo
├── *.mp4        — Intro video
└── *.pdf        — CV download
```

## ✨ Features
- 3D animated DNA helix (Three.js)
- Floating molecule particles
- Glassmorphism cards with 3D tilt
- Typing animation (hero title)
- Scroll-triggered GSAP animations
- Certifications auto-carousel
- Animated donut charts (research section)
- Skill progress bars
- Video showcase section
- CV download panel
- Fully responsive (mobile, tablet, desktop)
- Custom cursor
- Dark neon healthcare theme

## 🛠 Tech Stack
- Pure HTML5, CSS3, JavaScript (no build tools needed)
- Three.js r128 (CDN)
- GSAP 3.12 + ScrollTrigger (CDN)
- Google Fonts: Syne, DM Sans, JetBrains Mono
